# Welcome to Radio France Open Source

![Radio France](https://avatars.githubusercontent.com/u/17847038?v=4)

Radio France is dedicated to sharing our innovative projects and tools with the open-source community. Our repositories feature a diverse range of projects aimed at enhancing accessibility, improving web technologies, and providing robust solutions for developers.

## Featured Projects

### [DIB](https://github.com/radiofrance/dib)
An opinionated oci image builder. This project provides tools and configurations to build oci images efficiently and securely.

- **Language:** Go
- **License:** CeCILL Free Software License Agreement v2.1

### [RF-Liquidsoap](https://github.com/radiofrance/rf-liquidsoap)
Radio France's liquidsoap configuration with a self-contained docker infrastructure for resilient radio streaming. This project helps in setting up and managing robust radio streaming solutions.

- **Language:** Liquidsoap
- **License:** CeCILL Free Software License Agreement v1
